Hi, I'm Bert Timmerman, welcome here.

Another project related to gEDA is pcb-fpw, a footprint wizard for pcb, with this application one can create footprint files used by the pcb layout editor.
A GTK based version is pcb-gfpw, the name is derived from pcb and GTK footprintwizard.
There is also a Command Line Interface (CLI) version, it is named fpw which is derived from footprintwizard (current status: work in progress).
There is a fact file on the pcb-fpw github pages.
