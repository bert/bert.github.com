Hi, I'm Bert Timmerman, welcome here.

I have made a modest start with generating developer information for gEDA and pcb, the interactive layout editor.
The information is generated with Doxygen version 1.4.6.
The doxygen files for pcb can be found here.

Another project related to gEDA is pcb-gfpw, a GTK based footprint generator for pcb, the name is derived from GTK footprintwizard.
There is also a Command Line Interface (CLI) version, it is named fpw which is derived from footprintwizard (current status: work in progress).
With this application one can create footprint files used by the pcb layout editor.

The pcb-dxf HID is a DXF exporter to be used as a part of PCB and can be found here.

Then there are some pcb plug-ins in the pcb-plugins repository.
Some of the plug-ins I contributed are:
  • lockelements.c : to lock/unlock all elements in a pcb layout.
  • findelement.c : to find an element on the current pcb you are editing.
  • upth2pth.c : to change unplated holes to plated holes (Status: work in progress, so YMMV :-)
  • ghellopcb.c : to create a GTK dialog window showing a "Hello World" message (a proof of concept attempt)
And a lot of other plug-ins I collected from the gEDA mailing lists.
